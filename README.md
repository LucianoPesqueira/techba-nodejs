# Back-End Node JS

Contenido:

- Atender y responder solicitudes de servidor entregando datos a demanda en formato JSON.
- Configurar y consumir un esquema de modelo de datos en la nube basado en el servicio Firestore del proveedor Firebase.
- Conocer sobre estándares de seguridad y autenticacion.

Objetivos específicos:

1. Desarrollar aplicaciones web del tipo API REST.
2. Dominar patrones de arquitectura y comprender las diferencias entre cliente y servidor.
3. Utilizar Node.js y Express.js para crear aplicaciones backend.
4. Comprender el uso y gestión de paquetes, módulos nativos y de terceros.
5. Implementar soluciones con servicios dinámicos en la nube, como Firestore, y diferenciarlas de bases de datos tradicionales.
6. Implementar y desarrollar capas de autenticacion.
7. Integrar conocimientos adquiridos en un proyecto final que combine las tecnologías y herramientas aprendidas.
